# Автомобильный учет

### Используемые технологии
* Laravel
* Vue.js 3
* VueX
* Vuetify
* Axios
* Postgres


### Установка
Требуется создать базу данных Postgres и указать ее название, имя и пароль пользователя в /backend/.env
```
cd .\frontend\
npm install
```

### Запуск
1. Backend
```
cd .\backend\
php artisan serve
```
2. Frontend
```
cd .\frontend\
npm run serve
```
Сайт доступен по адресу http://localhost:8080/
